# Supplementary data for charge-altering histone mutations reprogram electrostatic coupling from single nucleosomes to higher-order chromatin structures

This repository contains supplementary data for "Charge-altering histone mutations reprogram electrostatic coupling from single nucleosomes to higher-order chromatin structures" by Peng al.

You can browse the following folders:

1.[mutation_dataset](./mutation_dataset): Including  Recurrent Mutation Set, Driver Mutation Set, and Combined Mutation Set mutation data.

2. [net_mutation](./net_mutation): match mutation data and cacluted DDG(mutation leading change binding free engrey).

3. [electrostatic_potential](./electrostatic_potential): Scripts and result files for calculated electrostatic potential.

4. [force](./force): Result files for calculated electrostatic force.

6. [MD](./MD): Scripts and results for molecular simulation.
