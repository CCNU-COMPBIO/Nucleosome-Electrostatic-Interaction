# Supplementary data for charge-altering histone mutations reprogram electrostatic coupling from single nucleosomes to higher-order chromatin structures

This repository contains supplementary data for "Charge-altering histone mutations reprogram electrostatic coupling from single nucleosomes to higher-order chromatin structures" by Peng et al.

You can browse the following folders:

1.[mutation_dataset](./mutation_dataset): Including  Recurrent Mutation Set, Driver Mutation Set, and Combined Mutation Set mutation data.

2.[net_mutation](./net_mutation): Contains matched mutation data and calculated ΔΔG values (mutation-induced changes in binding free energy).

3.[electrostatic_potential](./electrostatic_potential): Scripts and result files for electrostatic potential calculations.

4.[force](./force): Result files for electrostatic force calculations.

6.[MD](./MD): Scripts and results for molecular dynamics simulations.
